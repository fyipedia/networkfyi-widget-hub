FAQ

How many submarine cables does CableFYI document?

CableFYI documents 500+ active submarine cable systems with landing points, operators, length, capacity (Tbps), RFS year.

Which TLDs does TLDFYI cover?

TLDFYI covers all ICANN-recognized TLDs: ccTLDs, gTLDs, new gTLDs. Registry operator, WHOIS, DNSSEC, restrictions per TLD.

What IP data does IPFYI provide?

IPFYI provides geolocation (MaxMind GeoLite2), ASN lookup, CIDR ranges, IPv4/IPv6 conversion, reserved ranges.

What HTTP status codes does StatusCodeFYI cover?

StatusCodeFYI covers all RFC-registered codes (1xx-5xx) plus Cloudflare-specific (520-525, 530). Common debug scenarios per code.
